ES6 support back tick for multiple line string

the follow is working:

let multi = `
hello
world
my
name
is
asim`;
console.log(multi);

the follow is also working:

let name = "asim";
let multi2 = `
hello
world
my
name
is
${name}
`;
console.log(multi2);

Please open console to see the result

All browsers except IE support it
