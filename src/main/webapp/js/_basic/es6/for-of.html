ES6 fixed for in loop by using for of loop

The ES5 for in loop has a bug:

let array = [10,20,30];
let total = 0;
for (let index in array) {
	total = total + index;
}
console.log(total);  // output will be "0123"

Now use For of loop:


let array = [10,20,30];
let total = 0;
for (let value of array) {
	total = total + value;
}
console.log(total);  // output will be 60

The for-in loop is designed for iterating over an objects properties, like so:


var obj = {a:1,b:2};
for (let prop in obj) {
	console.log(prop);
}
// a
// b

Please always use for of on the array, set and map, always use for in on the object

All browsers except IE support it
