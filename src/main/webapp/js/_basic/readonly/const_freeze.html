compare const and Object.freeze

const lock the pointer, will throw exception when you change the pointer

	const foo = {};
	foo = {}; // TypeError: Assignment to constant variable.

Object.freeze lock the properties of an object

	let foo = Object.freeze({});
	foo.prop = 123;
	console.log(foo.prop) // undefined

changing frozen object properties will be silently ignored until you have 'use strict' in the top
