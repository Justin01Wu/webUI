test Let and Var difference

Let is blcok scope, var is function scope

the follow is working:

	{
		var x = "It's me"; 
	}
	console.log(x);  // will output "It's me"

the follow is NOT working:

	{
		let y = "It's me"; 
	}
	console.log(y);  // will throw ReferenceError

Please open console to see the result
