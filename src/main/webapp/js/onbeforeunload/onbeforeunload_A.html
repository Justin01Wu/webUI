Chrome behaviour:

  • please open console and enable logs between pages like Chrome "preserve log"
  • onbeforeunload event will be triggered when page is left or closed
  • onbeforeunload event will be triggered when frame is left
  • onbeforeunload event will be triggered when frame is reloading
  • onbeforeunload event will NOT be triggered when frame is closed

IE 11 behaviour:

  • please open console and enable logs between pages like Chrome "preserve log"
  • onbeforeunload event will be triggered when page is left or closed
  • onbeforeunload event will be triggered when frame is left
  • onbeforeunload event will be triggered when frame is reloading
  • onbeforeunload event will be triggered when frame is closed
